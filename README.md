# Чотири квадрати

Адаптивний офлайн-PWA для дитячої гри з чотирма фізичними поворотними плитками 2×2.

## Запуск

Потрібен Node.js 20+.

```bash
npm run serve
```

Відкрийте `http://localhost:4173`. Тести: `npm test`.

## GitHub Pages

Workflow `.github/workflows/pages.yml` тестує та публікує застосунок після push у `main`. У репозиторії оберіть **Settings → Pages → Source: GitHub Actions**.

## Дані й приватність

Усі 256 схем генеруються локально. Прогрес, налаштування і поточна сесія зберігаються лише в `localStorage`; акаунтів і сервера немає.

Складності зараз визначаються детермінованим евристичним ранжуванням і містять по 64 схеми. Функція класифікації ізольована в `src/domain.js`, тому затверджений педагогічний каталог можна підставити без змін решти застосунку.
